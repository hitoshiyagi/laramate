# Laramate  
Laravel 命名支援ツール

---

## 概要
**Laramate** は、Laravel 開発における  
「命名ルールの検討」や「定型コード作成」にかかる時間を削減するための  
**命名要素自動生成・管理アプリ**です。

CRUD 作成や機能追加のたびに発生する  
「この名前どうする？」という迷いを減らし、  
実装に集中できる環境を作ることを目的としています。

---

## 開発背景
Laravel の学習および実務開発を進める中で、以下の課題を感じました。

- テーブル名・モデル名・コントローラ名などを毎回考える負担  
- 命名の揺れによる可読性・保守性の低下  
- チーム開発における命名ルールの不統一  

特に CRUD 開発では、  
**命名を考える時間が実装スピードを大きく下げている** と感じたため、  
命名工程そのものを支援するアプリとして Laramate を開発しました。

---

## 主な機能
- Laravel の命名規則に沿った要素名の自動生成  
- CRUD 前提の命名セット管理  
- プロジェクトごとの命名ルール整理・再利用  
- JavaScript による即時生成・表示  

---

## こだわった点
- Laravel の公式命名規則・慣習をベースに設計  
- 初学者でも「なぜこの名前になるのか」が理解できる構造  
- チーム開発でも使える再現性・一貫性を重視
