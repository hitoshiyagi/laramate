# laramate
Laravel プロジェクトの構成要素（モデル名・コントローラ名・DB名・テーブル名 等）を自動生成し、さらに “開発のための手順書” までワンクリックで作成できる、Laravel 初学者向けの生産性向上ツールです。

## 一言で言うと
**「Laravel プロジェクトの命名・準備作業を自動化するアプリ」**  
プロジェクト生成の初動が一気に早くなります。

# 開発の動機と課題設定
Laravel を使って個人開発を進める中で、以下の問題に直面しました。

- プロジェクト開始時の命名作業（モデル / テーブル / DB / コントローラ）が毎回面倒  
- 命名のルールを調べる時間が無駄  
- 作成後の“やるべき手順”を毎回忘れる  

特に、**命名規則（複数形 / キャメル / パスカル）を覚えきれない問題**が初学者にとって大きなストレスでした。

### 課題
1. 毎回検索が必要な命名ルール  
2. プロジェクト作成後に何をすべきか分かりにくい  
3. チーム開発で命名が統一されないことによる可読性の低下

---

# 解決策
laramate では、以下の仕組みで課題を解消しました。

### 自動命名エンジン（Vanilla JS）
- キーワードを入力するだけで  
  **モデル名 / テーブル名 / DB 名 / コントローラ名** を瞬時に生成
- Laravel の命名規則に基づき  
  - PascalCase  
  - snake_case  
  - plural 変換  
  を自動で処理

### 手順書の完全自動生成
Laravel プロジェクトで次に行うべきコマンドやファイル編集を  
**カード形式＆コピー可能なコードブロック**で表示。

これにより、  
「次に何をすればいいの？」  
という悩みを完全にゼロに。

### UX 改善
- AdminLTE カードで読みやすく整理  
- コード部分は `pre` で整形  
- コピー機能付き  
- 生成後は自動スクロールで画面移動  
- バリデーション（半角英数字のみ）も全フォームに適用

# 主な機能

## 1. 要素名の自動生成（キーワード → 命名セット）
例）`project`  
- Model: `Project`  
- Controller: `ProjectController`  
- DB: `project_db`  
- Table: `projects`  

## 2. プロジェクト作成フローの手順書生成
- `laravel new`  
- `.env` 設定  
- マイグレーション生成  
- コントローラ作成  
- モデル作成  
などをカード形式で表示。

## 3. コードブロックのコピー機能  
FontAwesome アイコン付き。

## 4. 入力値バリデーション  
- 全フォームで  
  `^[a-zA-Z0-9]+$` の半角英数字バリデーション  
- バリデーションエラーはフォーム直下に表示。

# 技術的な工夫

### ■ JavaScript での命名規則変換
- 単語の複数形変換  
- PascalCase / snake_case  
- Laravel のリソース名に沿った自動変換ロジック

### ■ Laravel のセキュリティ対策
- CSRFトークンを Fetch API に自動付与  
- バリデーションを Controller でしっかり実装  
- XSS 対策として Blade のエスケープ利用

### ■ AdminLTE をフル活用した UI
- カード間の余白
- 黒背景のコードヘッダー  
- ダークグレーのコードブロック  
- コピー完了アニメーション
